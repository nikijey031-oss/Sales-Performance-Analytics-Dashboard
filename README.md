# Sales Performance Analytics Dashboard

## Overview
An interactive Tableau dashboard built to analyze sales performance and identify business insights from sales data.

## Key Insights
- Total Orders: 4,922
- Total Sales: 2,261,537
- Monthly sales trend analysis
- Top 20 cities by sales
- Business-oriented data visualization

## Tools Used
- Tableau
- Microsoft Excel

## Dashboard Features
- KPI Cards
- Monthly Sales Trend Analysis
- City-wise Sales Performance Analysis
- Interactive Filtering
